Bombardier UIS

Bombardier UIS is a ERP application, which is mainly targeting for railway workers on MTA Long Island Rail Road (LIRR) and MTA Metro-North Railroad (MNR) railroad in USA. This application is used to manage temporary speed restriction (TSR) on train in a particular territory where the workers working on. Application can able communicate other component like STS, DMS, RWPS, NMS using TCP socket communication. Users can able to create TSR, Configure the application based on the role.

Client: Bombardier, USA
